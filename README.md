# DFF Design Lab

A collection of Verilog RTL designs built using D Flip-Flops.

This repository is part of my journey in learning Digital Design, RTL Design, and VLSI Front-End Design.

## Topics Covered

### D Flip-Flops
- Basic D Flip-Flop
- D Flip-Flop with Synchronous Reset
- D Flip-Flop with Asynchronous Reset
- D Flip-Flop with Enable
- D Flip-Flop with Reset and Enable

### Registers
- 4-bit Register
- 8-bit Register
- Parameterized Register

### Counters
- MOD-3 Counter
- MOD-5 Counter
- MOD-10 Counter
- Ring Counter
- Johnson Counter

### Shift Registers
- Serial-In Serial-Out (SISO)
- Serial-In Parallel-Out (SIPO)
- Parallel-In Serial-Out (PISO)
- Universal Shift Register

### FSMs
- Sequence Detector
- Traffic Light Controller
- Other FSM-based Designs
